\begin{exercise}
The Intermediate Value Theorem states:
If $f$ is a continuous function for all $x$ in the closed interval
$[v,n]$ and $d$ is between
\wordChoice{\choice{v}\choice[correct]{f(v)}} and
\wordChoice{\choice{n}\choice[correct]{f(n)}}, then there is a
number \wordChoice{\choice[correct]{m}\choice{f(m)}} in
  \wordChoice{\choice{[f(v),f(n)]}\choice[correct]{[v, n]}} such that
  \wordChoice{\choice[correct]{f(m) = d}\choice{f(d) = m}}.
