\begin{exercise}


Consider the curve given as the graph of the function $y=3x^5-\frac{1}{x}$ on $(0,3]$. 

Find a parametrization for this curve. 


\begin{align*}
x(t)&=\answer{t}\\
y(t)&=\answer{3t^5-\frac{1}{t}}
\end{align*}

for $\answer{0} < t \leq \answer{3} $. 


\end{exercise}
