\begin{exercise}

The curve $C$ is given by the following set of parametric equations:

\[ \begin{cases}
x(t)=   -4e^{2t} \\
y(t)=   5 - e^{4t}
\end{cases}
, -\infty < t < \infty \]

Eliminate the parameter and find a Cartesian representation of the given curve.
\[
y= \answer{ 5 -\frac{1}{16}x^2}
\]
(Write out $y$ in terms of $x$.)

\begin{hint}
From the equation for $x(t)$, we find $e^{2t} = \answer{-\frac{1}{4}x}$.  

\begin{question}
Note that $y=5-\left(e^{2t}\right)^{\answer{2}}$
\end{question}
\end{hint}

\begin{exercise}
Which of the following is the graph of the curve $C$?
\end{exercise}

\begin{multipleChoice}

          \begin{tikzpicture}

\choice[correct]{\begin{image}

	    \begin{axis}[

As $t$ increases, the curve is traced out:

\begin{multipleChoice}
\choice{left to right}
\choice[correct]{right to left}
\end{multipleChoice}

Note that as $t$ increases, $e^{2t}$ increases, so $x(t) = -4e^{2t}$ becomes increasingly negative!
\end{exercise}
