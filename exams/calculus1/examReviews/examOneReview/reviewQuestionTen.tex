$$\ln(x)\leq f(x) \leq x-1, x\neq 1, 0<x<7$$

Select the correct limit, and justification:

$\lim_{x\to 1} f(x)=$ 
\begin{prompt}
\begin{multipleChoice}
\choice{1, using the Intermediate Value Theorem}

\choice{Not enough information to determine the limit}
\end{multipleChoice}
\end{prompt}
